# HandAI boot artwork v2

`handai-boot-art-v2.png` was generated specifically for HandAI with OpenAI's
built-in image generation tool on 2026-07-23. It is not copied from another
game, firmware, artist, or stock-asset collection.

Creative direction: a futuristic pixel-art RG35XXSP-style handheld acting as an
AI command cockpit, projecting cyan/magenta neural links through a portal to a
constellation of remote computers. The generated image intentionally contains
no text; `generate-bootlogo.py` adds the exact `HANDAI / REMOTE AI COCKPIT`
bitmap title and writes the result into the device's required 640x480x32 BMP
container.

The `.bmp` file is a nearest-neighbour 640x480 build source derived from the
full-resolution PNG. Both files are project assets covered by the repository's
license terms.
