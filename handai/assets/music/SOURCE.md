# Music asset sources

All compositions in this directory are by **Zane Little Music** and published
under [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/).
The files bundled by HandAI are mono 22.05 kHz PCM conversions made for the
RG35XXSP. No generative-AI music is included.

| HandAI file | Composition | Source page |
|---|---|---|
| `01-pocket-signal.wav` | Flowerbed Fields | https://opengameart.org/content/flowerbed-fields-loop |
| `02-copper-constellation.wav` | Apple Cider | https://opengameart.org/content/apple-cider |
| `03-green-compile.wav` | Pixel Sprinter (loop) | https://opengameart.org/content/pixel-sprinter |
| `04-wingbeat-relay.wav` | The Cool Factor (loop) | https://opengameart.org/content/the-cool-factor |
| `05-blue-brackets.wav` | Heroic Loop | https://opengameart.org/content/heroic-loop |
| `06-crimson-pincers.wav` | Void Estate | https://opengameart.org/node/153209 |

The old filenames are retained to avoid breaking existing installations and
cached preferences; the UI displays the real composition titles.

Run `python scripts/import-cc0-chiptunes.py` to download the exact upstream
assets and rebuild these normalized copies.
